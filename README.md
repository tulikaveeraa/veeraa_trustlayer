# veeraa_trustlayer
Code for Veeraa's Trust Layer processes

Author: Tulika Jain

Date: May 2026

Purpose: This repository contains codes for various steps of the Trust Layer process. These include: 

	(1) Managing our community database
	(2) Phone Survey Tracking
	(3) Data Cleaning and Analysis

Instructions: 

	(1) For each cycle of the Trust Layer process, there is a different folder with Month-YY name. 
	(2) Commit Code -> Pull code -> Review Code -> Make changes -> Commit Code
	(3) Once one cycle is complete, the code and the output cannot be amended. 
	(4) When a new cycle starts, create a new folder and copy any relevant old code for reuse in workflow
